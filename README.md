# c++ demo project with mac qt

最简单的 qt c++项目!!!



## 开发环境: 
Mac 10.15.7
Qt 5.9.9
c++ 
c



## 目录说明

- bin 为项目编译后的 Mac应用程序

- src 为项目源文件



## 头文件说明

```
#include <QFileDialog>   QT弹窗对象
#include <QString>      QT字符串对象
#include <QTextCodec>   QT文本编码对象
#include <QDebug>       QT调试对象
#include <string.h>    C字符串处理库
```
